## [1.0.0] - 2026-04-20

### Added

- 项目初始化：基于 Electron + React + TypeScript。
- 播放能力：支持播放/暂停/上一曲/下一曲/进度控制。
- 音源接入：集成网易云音乐 API，支持在线歌曲播放。
- 歌词与视觉：实现动态歌词与背景色提取。
- 页面模块：首页推荐、音乐排行榜、歌手详情、全屏播放页。
- 主题系统：支持深色/浅色模式切换。
- 数据持久化：用户配置与播放记录本地存储。
- 打包产物：提供 Windows 安装版与便携版。
